## Gestures
- **Index finger**: Draw
- **Index + Middle finger**: Switch mode (Object/Math)
- **3 fingers up**: Clear canvas
- **Closed fist**: Erase
- **Press 'h'**: Toggle help
- **Press 'q'**: Quit
